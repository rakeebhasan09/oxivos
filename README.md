# 👗 Fashion Store

A modern and responsive fashion e-commerce frontend built with **Next.js**, **JavaScript**, **Tailwind CSS**, and **shadcn/ui**. The project focuses on delivering a fast, elegant, and user-friendly shopping experience with clean UI and optimized performance.

## ✨ Features

- 🛍️ Modern and responsive design
- 📱 Mobile-first layout
- 🔍 Product search and filtering
- 🏷️ Product categories
- ❤️ Wishlist functionality
- 🛒 Shopping cart
- 👤 Authentication pages
- 📄 Product details page
- 🌙 Beautiful and reusable UI components
- ⚡ Server-side rendering (SSR) & Static Site Generation (SSG)
- 🚀 Optimized performance with Next.js

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- React 19
- JavaScript
- Tailwind CSS
- shadcn/ui
- Lucide React
- Framer Motion
- Axios
- TanStack Query
- React Hook Form

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/fashion-store.git
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser.
